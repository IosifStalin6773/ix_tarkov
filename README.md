# ix_tarkov



# [ENG] Project Desc
This repository contains a scheme inspired by Escape From Tarkov (EFT) implemented on the Helix (ix) framework for the game Garry's Mod (GMod). The goal is to recreate mechanics and elements characteristic of EFT's inventory and object management system within the Helix roleplay environment, offering a more immersive and realistic experience.

# Main Features

 - EFT-style inventory system with grid and advanced object management.
 - Full integration with Helix for compatibility with other plugins and schemas.
 - Support for customizable items, including weapons, equipment, and consumables.
 - Intuitive visual interface inspired by Escape From Tarkov.
 - Modular base for future expansions and improvements.

# Goal
To provide the GMod community with a unique schema that combines the hardcore gameplay of EFT with the flexibility of the Helix framework, ideal for servers focused on survival, trade, and tactical combat.





# [ESP] Descripción del proyecto
Este repositorio contiene un esquema inspirado en Escape From Tarkov (EFT) implementado en el marco Helix (ix) para el juego Garry's Mod (GMod). El objetivo es recrear la mecánica y los elementos característicos del sistema de gestión de objetos e inventario de EFT dentro del entorno de juego de rol Helix, ofreciendo una experiencia más inmersiva y realista.

# Características principales

 - Sistema de inventario al estilo EFT con cuadrícula y gestión avanzada de objetos.
 - Integración completa con Helix para garantizar la compatibilidad con otros complementos y esquemas.
 - Compatibilidad con objetos personalizables, incluyendo armas, equipamiento y consumibles.
 - Interfaz visual intuitiva inspirada en Escape From Tarkov.
 - Base modular para futuras expansiones y mejoras.

# Objetivo
Proporcionar a la comunidad GMod un esquema único que combine la jugabilidad hardcore de EFT con la flexibilidad del marco Helix, ideal para servidores centrados en la supervivencia, el comercio y el combate táctico.
